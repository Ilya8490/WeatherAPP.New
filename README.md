# SkyCast Weather

A production-ready weather application scaffold built with React, TypeScript, Tailwind CSS, and TanStack Query.

## Stack

- React + TypeScript + Vite
- Tailwind CSS
- TanStack Query
- WeatherAPI.com
- localStorage for favorites and theme persistence

## Setup

1. Copy `.env.example` to `.env`
2. Add your WeatherAPI key as `VITE_WEATHER_API_KEY`
3. Install dependencies with `npm install`
4. Start the app with `npm run dev`
